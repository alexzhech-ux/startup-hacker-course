# День 2. Основы Vue и реактивности

Темы:
- создание Vue-приложения  
- реактивные переменные  
- computed-свойства  
- интерполяция текста в шаблоне.  

## Практическое задание

В этом уроке ты создашь свой рейтинг любимых книг.  

Цель задания:
- научиться выводить списки в шаблоне vue  
- освоить работу с реактивными переменными  

## Порядок выполнения

1. Сделай форк учебного репозитория в github https://github.com/rsukhar/startup-hacker-course#    
2. ! каждое следующее задание нужно выполнять в папке day_N, где N-номер дня стажировки. Т.к. задания построены вокруг одного проекта, код, написанный в прошлый день нужно будет переносить в новую папку.  
3. в App.vue объяви реактивную переменную со списком книг (массив с 3-5 объктами)  
4. объекты должны хранить следующую информацию:  
 - название  
 - описание  
 - жанр  
 - обложка книги  
 - пометка 18+ (да/нет)
 - рейтинг  
5. выведи данные в виде карточек с помощью v-for (не забудь добавить атрибут key)  
6. для каждой карточки добавь рейтинг в звездочках (от 1 до 5).
